Group Travel · Planning App

Travely

Simplifying group travel planning in one place

Role UX/UI Designer
Tools Figma · Maze
Year 2025
Overview

Travely is a mobile app designed to simplify group travel planning by allowing users to organize itineraries, manage shared expenses, and coordinate trips with friends or family in one place. The goal was to reduce the friction and confusion that often happens when multiple people are involved in planning a trip.

Tools & Technologies

Figma Maze
The Challenge
"Planning trips with multiple people can quickly become chaotic. Users often struggle with tracking shared expenses, coordinating itineraries, managing group decisions, and keeping all trip information in one place."
  • Tracking shared expenses across the group
  • Coordinating itineraries with multiple people
  • Managing group decisions and communication
  • Keeping all trip information centralized
Research

Understanding the user

Methods

  • User interviews about group travel experiences
  • Common pain points in trip coordination
  • Existing travel and expense apps analysis
  • Usability testing with Maze

Key Insights

  • Users rely heavily on multiple apps (notes, chats, spreadsheets)
  • Expense tracking is one of the biggest frustrations
  • Users want a centralized and visual way to organize trips
Design Process

Steps

  1. Defined user flows for planning, expenses, and itinerary
  2. Created wireframes to structure key features
  3. Designed high-fidelity UI in Figma
  4. Built a consistent design system

Special Focus

  • Clear navigation
  • Easy interaction between users
  • Visual clarity for shared information
The Solution

A centralized experience

Travely offers a centralized platform where users can create and manage trip itineraries, track shared expenses in real time, invite and collaborate with friends, and organize travel details in one place. The interface uses a clean, modern style with travel-inspired visuals.

  • Create and manage trip itineraries
  • Track shared expenses in real time
  • Invite and collaborate with friends
  • Organize travel details in one place
  • Chat integration for coordination
